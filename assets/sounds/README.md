# Sound Files

Place the following sound files in this directory:

- `chime.mp3` - Timer start sound
- `complete.mp3` - Timer finish/completion sound
- `pop.mp3` - Bubble wrap pop sound
- `drop.mp3` - Water ripple sound
- `click.mp3` - Button press sound (optional)

All files should be in MP3 format and optimized for mobile use.
